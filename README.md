### AT ANULAR UIKIT

献给喜欢ANGULAR的小伙伴

install

```
npm install at-ng
```

add the module to your root module

```
AtModule.forRoot()

```
Add the AT-UI In your project

in angular-cli project

.angular-cli.json

```
"styles": [
        "styles.css",
        "../node_modules/at-ui-style/css/at.css"
      ],
```
