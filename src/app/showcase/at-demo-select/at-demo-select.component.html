select

基本用法，可添加 disabled 属性禁用选择器

设置 multiple 属性可开启多项选择器，此时绑定的 model 将接受数组类型的数据

可设置 size 属性控制选择器大小，提供三种尺寸：large，normal，small

设置 clearable 属性可清空已选项

设置 tagAble 属性可以输入(仅限多选)

设置 searchable 进行下拉框搜索
参数 说明 类型 可选值 默认值
ngModel 指定当前组件的 value 值，可通过 ngModel双向 绑定 String / Number / Array - -
multiple 是否支持多选 Boolean - false
disabled 是否禁用选择器 Boolean - false
allowClear 是否支持清空功能 Boolean - false
searchable 是否支持搜索功能 Boolean - false
atPlaceholder 选择器的占位文案 String - 请选择
atSize 设置选择器的尺寸 String large, normal, small normal
参数 说明 类型 可选值 默认值
disabled 是否禁用选择器 Boolean - false
atLabel 标题 string | number - -
atValue 对应的值 string | number - -
参数 说明 类型 可选值 默认值
change 更新选中值都后进行回调 - - -
