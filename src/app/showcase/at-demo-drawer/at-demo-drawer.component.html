参数 说明 类型 默认值
[atClosable] 是否显示右上角的关闭按钮 boolean true
[atMaskClosable] 点击蒙层是否允许关闭 boolean true
[atMask] 是否展示遮罩 boolean true
[atMaskStyle] 遮罩样式 object -
[atBodyStyle] Drawer body 样式 object -
[atTitle] 标题 stringTemplateRef<-> -
[atVisible] Drawer 是否可见 boolean -
[atPlacement] 抽屉的方向 'top''right''bottom''left' 'right'
[atWidth] 宽度, 只在方向为 'right'或'left' 时生效 numberstring 256
[atHeight] 高度, 只在方向为 'top'或'bottom' 时生效 numberstring 256
[atOffsetX] x 坐标移量(px), 只在方向为 'right'或'left' 时生效 number 0
[atOffsetY] y 坐标移量(px), 高度, 只在方向为 'top'或'bottom' 时生效 number 0
[atWrapClassName] 对话框外层容器的类名 string -
[atZIndex] 设置 Drawer 的 z-index number 1000
(atOnClose) 点击遮罩层或右上角叉或取消按钮的回调 EventEmitter<MouseEvent> -
