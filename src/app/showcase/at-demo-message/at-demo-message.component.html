Message

相比 Notification，Message 更轻量，居中显示在页面顶部，用于展示全局消息，例如操作的反馈信息

  • 提供消息、成功、错误、警告等反馈提示
  • 在顶部居中显示，并自动消失，是一种不打断用户操作的轻量级提示

在 组件 中注入 message_service，我们可以直接通过 this.message_service 操作实例

  • this.message_service(config)
  • this.message_service.info(config)
  • this.message_service.success(config)
  • this.message_service.warning(config)
  • this.message_service.error(config)
  • this.message_service.loading(config)
参数 说明 类型 可选值 默认值
type 通知提醒的类别 String success, error, warning, info info
message 通知的内容 String - -
duration 自动关闭的延时，单位为毫秒 Number - 4000
