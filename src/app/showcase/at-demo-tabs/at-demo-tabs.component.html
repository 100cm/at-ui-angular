tabs
参数 说明 类型 默认值
[atIndex] 当前激活 tab 面板的 序列号，可双向绑定 number 无
[atAnimated] 是否使用动画切换 Tabs，在 atTabPosition=top丨bottom 时有效 boolean丨<inkBar:boolean, tabPane:boolean> true, 当 type="card" 时为 false
[atSize] 大小，提供 largedefault 和 small 三种大小 string 'default'
[atTabBarExtraContent] tab bar 上额外的元素 TemplateRef<void> 无
[atTabBarStyle] tab bar 的样式对象 object -
[atTabPosition] 页签位置，可选值有 toprightbottomleft string 'top'
[atType] 页签的基本样式，可选 line、card 类型 string 'line'
[atTabBarGutter] tabs 之间的间隙 number 无
[atHideAll] 是否隐藏所有tab内容 boolean false
[atShowPagination] 是否超出范围时显示pre和next按钮 boolean true
(atIndexChange) 当前激活 tab 面板的 序列号变更回调函数 EventEmitter<number> 无
(atSelectChange) 当前激活 tab 面板变更回调函数 EventEmitter<<atIndex: number,tab: NzTabComponent>> 无
(atOnNextClick) next 按钮被点击的回调 EventEmitter<void> 无
(atOnPrevClick) prev 按钮被点击的回调 EventEmitter<void> 无
参数 说明 类型 默认值
[atTitle] 选项卡头显示文字 string ｜ TemplateRef<void> -
[atForceRender] 被隐藏时是否渲染 DOM 结构 boolean false
[atDisabled] 是否禁用 boolean -
(atClick) title被点击的回调函数 EventEmitter<void> -
(atSelect) tab被选中的回调函数 EventEmitter<void> -
(atDeselect) tab被取消选中的回调函数 EventEmitter<void> -
