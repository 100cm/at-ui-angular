Notification 提醒

在页面的右上角全局显示通知提醒信息，跟 Message 有点类似。常用于以下场景：

  • 较复杂的通知内容
  • 有交互的通知
  • 系统的推送

注入 AtNotificationService


  • this.notification_service.show(config)
  • this.notification_service.success(config)
  • this.notification_service.error(config)
  • this.notification_service.warning(config)
  • this.notification_service.info(config)

实例config接收如下参数：


  • type - 通知提醒的状态
  • title - 消息标题
  • message - 消息内容
  • duration - 自动关闭的延时
  • showClose - 是否显示关闭按钮
  • icon - 自定义消息提醒的图标
参数 说明 类型 可选值 默认值
type 通知提醒的类别 String success, error, warning, info info
title 必填，通知的标题 String - -
message 通知的内容 String - -
duration 自动关闭的延时，单位为毫秒 Number - 4000
showClose 是否显示关闭按钮 Boolean - true
icon 自定义消息提醒的 ICON String - -
