Carousel
参数 说明 类型 默认值
[atAutoPlay] 是否自动切换 boolean false
[atAutoPlaySpeed] 切换时间(毫秒)，当设置为0时不切换 number 3000
[atDots] 是否显示面板指示点 boolean true
[atDotRender] Dot渲染模板 TemplateRef<{{' $implicit: number'}}> -
[atEffect] 动画效果函数，可取 scrollx, fade string scrollx
[atVertical] 垂直显示 boolean false
(atAfterChange) 切换面板的回调 EventEmitter<number> 无
(atBeforeChange) 切换面板的回调 EventEmitter<{{ 'from: number; to: number' }}> 无
[atEnableSwipe] 是否支持手势划动切换，仅在自行引入 hammer.js 的情形下生效 boolean true
