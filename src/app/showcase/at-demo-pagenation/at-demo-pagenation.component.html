分页
参数 说明 类型 可选值 默认值
atPageIndex 当前页码 Number - 1
total 总的数据条目数 Number - 0
atPageSize 每页最多展示的条目数 Number - 10
atPageSizer 是否显示每页展示数量的 select 选择框 Boolean - false
atQuickJump 是否显示快速跳转 Boolean - false
size 分页组件的大小 String small -
simple 是否为极简风格 Boolean - false
事件名称 说明 返回值
pageIndexChange 页码改变时触发的回调 页码
pageSizeChange 切换每页显示的条数时触发的回调 -
