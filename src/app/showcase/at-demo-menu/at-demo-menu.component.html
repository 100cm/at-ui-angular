Menu 导航菜单

导航菜单一般分为顶部导航和侧边导航，是为页面提供导航功能的菜单列表

顶部导航需要设置 mode="horizontal"

可以在 atSubmenu 中嵌套 atSubmenu 组合成多级菜单

导航菜单默认 atType="inline",并且itemGroup 外层需要多包裹一层 inlineMenu
参数 说明 类型 可选值 默认值
atMenuType 导航菜单的模式 String inline, horizontal, vertical inline
atMenuType 导航菜单的主题 String light, dark light
参数 说明 类型 可选值 默认值
isOpen 是否打开 Boolean - false
