Modal 模态框

模态对话框，当需要询问用户处理事务，又不希望跳转页面时，可以使用模态框 Modal 在当前页面打开一个浮层并承载相应的操作。

当需要弹出一个简洁的确认框时，也可以使用默认的精简版模态框。 atModalService，可以直接通过 this.atModalService. 对象操作实例方法

Modal 实例方法

通过调用 this.atModalService. 的方法来使用：


  • this.atModalService.modal(config)
  • this.atModalService.info(config)
  • this.atModalService..success(config)
  • this.atModalService..warning(config)
  • this.atModalService..error(config)
参数 说明 类型 可选值 默认值
show 是否显示 boolean - -
showHeader 是否显示头部 boolean - -
showFooter 是否显示底部 boolean - -
atType 模态框类型 string normal | confirm normal
top 距离顶部的距离 Number - -
width 宽度 number - 520
onCancel 点击取消/关闭的回调 Function - -
onOk 点击确认的回调 Function - -
footer 底部选择器指令 Directive - -
header 头部选择器指令 Directive - -
body 内容选择器指令 Directive - -
参数 说明 类型 可选值 默认值
message 消息体 string | TemplateRef<void> - -
showHeader 是否显示头部 boolean - -
showFooter 是否显示底部 boolean - -
atComponent 动态模板 TemplateRef<void> - -
title 标题 string | TemplateRef<void> - -
top 距离顶部的距离 Number - -
width 宽度 number - 520
atOnClose 点击取消/关闭的回调，返回 false 时不关闭 (Function () => false | void) | Promise<void | false > - -
atOnOk 点击确认的回调 ,返回 false 时不关闭 Function () => true | false - -
atAfterClose 关闭后的回调 Function () => void - -
